⚙️ ตั้งค่า Google Drive

ยังไม่ได้เชื่อมต่อ Google Drive
ต้องสร้าง Client ID จาก Google Cloud Console ก่อน — เปิด Google Cloud Console ↗
📋 วิธีสร้าง Client ID (คลิกเพื่อดู)
  1. เปิด Google Cloud Consoleไปที่ console.cloud.google.com → สร้าง Project ใหม่หรือเลือก Project เดิม
  2. เปิด Google Drive APIไปที่ APIs & Services → Library → ค้นหา "Google Drive API" → Enable
  3. สร้าง OAuth Consent ScreenAPIs & Services → OAuth consent screen → เลือก External → กรอกชื่อแอป → Save
  4. สร้าง CredentialsAPIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID → Web application
  5. ใส่ Authorized originsเพิ่ม URL ที่ใช้เปิดไฟล์ HTML เช่น https://username.github.io หรือ http://localhost
  6. คัดลอก Client IDจาก Credentials แล้ววางในช่องด้านบน
✅ กำหนดค่าโฟลเดอร์ "รายงานประจำวัน" ไว้แล้ว ไฟล์จะถูกส่งไปที่โฟลเดอร์นี้อัตโนมัติ หากต้องการเปลี่ยนให้วาง Folder ID ใหม่

🗑 รีเซ็ตข้อมูลทั้งหมด

การรีเซ็ตจะลบข้อมูลน้ำฝนทั้งหมดที่กรอกไว้ในอุปกรณ์นี้ รวมถึงข้อมูลสะสมทุกวัน ไม่สามารถกู้คืนได้

⚠️ แนะนำให้ สำรองข้อมูล (JSON) ก่อนรีเซ็ต เผื่อต้องการกู้คืนภายหลัง
